🛡️ Admin Promotion Info

How to promote or manage other system admin accounts.

👑 Managing Roles in MongoDB Atlas:

To promote another student profile to an administrator, look up the student by their email inside the MongoDB Atlas users collection, and set their role field string value to:

role: "admin"

Note: Demoting or deleting the final admin account from this admin portal is strictly protected by the system.
